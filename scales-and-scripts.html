Playing sound on Linux — one-time setup

1. Install the library: pip install music21

2. Install a MIDI player + soundfont (Ubuntu/Debian): sudo apt install timidity fluid-soundfont-gm

3. The course writes MIDI files with stream.write('midi', fp='riff.mid'), then you play them from the terminal: timidity riff.mid

Prefer a graphical player? VLC and DAWs like Ardour or Reaper open .mid files too. And when you want live, instant sound later on, FoxDot and Sonic Pi both run beautifully on Linux — we point at them in Part IV.
